### 바인드변수
바인드변수란? sql에서 쿼리 실행 시 동적으로 값을 지정할 수 있도록 하는 변수

- 쿼리 작성 시 값을 직접 입력하는 대신 변수를 사용해서 값을 바인딩하는 방식
- 쿼리의 성능을 최적화하고 보안성을 높이기 위해 사용 (데이터베이스 성능 향상!)

- SQL 실행 속도 향상 바인드 변수를 사용하면 동일한 SQL 문을 여러 번 실행할 때 하드 파싱(Hard Parsing) 과정을 줄일 수 있어요. 데이터베이스는 기존 실행 계획을 재사용하므로 속도가 향상됩니다.

- SQL 인젝션 방지 사용자 입력을 직접 SQL에 포함하면 공격자가 SQL 인젝션을 수행할 위험이 있지만, 바인드 변수를 사용하면 문자열을 안전하게 처리할 수 있어 보안성이 높아집니다.

- 메모리 사용 최적화 데이터베이스가 쿼리를 캐싱하여 동일한 구조의 SQL 문을 효율적으로 실행할 수 있어 메모리 사용을 줄이고 성능을 최적화할 수 있어요.

### 해시
